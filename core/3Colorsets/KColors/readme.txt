Config: Core

Colorsets is to configure the color of the various parts of Fvwm such as window decor, menus, modules, etc. This list includes horizontal and vertical gradients.
https://www.fvwm.org/Wiki/Config/Colorsets

Check core/2Colorsets/FVWM-Colorsets.def (Colorsets 1 to 14)

To install or edit .fvwm/userExt.sys

## Add the below line below "2.Colorsets / Core".
Read $[CORE_DIR]/3Colorsets/KColors/kColorsets.sys
